Web App for Matting by Steven Zhu

This is a small project for my software engineering course. This project is completed by Liao Yiming and me.

In this web app, you can upload a portrait photo and get a matting result by drawing a Trimap on the page.

Note that this is just a course site and you can not use it for any commercial purpose. Just try it for fun.


Purpose

The purpose of this small project is that we want to provide a simple and light app for people to get a matting result of their protrait. People do not need to download and install any software just in order to get a matting of their photo.

Algorithm

We used the algorithm of KNN Matting (Project Site) to implement our project.

How to use

You just need to use the link below to head to the project site and follow these steps:

1. Upload a portrait and click (your photo should not beyond resolution of 320*320 or size of 2M).

2. Draw a Trimap in the middle Trimap section and click Convert.

3. Wait for several seconds and see the result.

4. If the result does not show correctly, please clear your browsing data and refresh the page.
