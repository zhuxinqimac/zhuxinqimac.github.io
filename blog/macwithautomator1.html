Mac with Automator : Part 1 by Steven Zhu

Have you ever complaint that you cannot copy and paste the directory path when using macOS? Have you ever opened your complex photoshop but only to give a filter to your image? I guess so many times that you camplaint about repeating simple tasks but lots of time being cost.

However Automator is used to solve this problem.

I'm gonna use several simple samples to show you how helpful Automator can be on macOS. However in this article I'll cover only one example (the other two can be found in next blog).



Introduction

Automator is an application developed by Apple Inc. for macOS that implements point-and-click (or drag and drop) creation of workflows for automating repetitive tasks into batches for quicker alteration, thus saving time and effort over human intervention to manually change each file separately.

For more infomation, please head to https://en.wikipedia.org/wiki/List_of_OS_X_components#Automator.



Sample

I'll use three small samples to illustrate how Automator works(in this article I'll cover only one, the other two will be in next blog). If you use Windows, I guess this software Jitbit Macro Recorder may help you, but I haven't tried it myself.

The three samples show: Copy and reveal the path of a folder/file in Finder, Save and clean the Desktop, and Add a filter to an image.


Copy and reveal the path of a folder/file in Finder

Open Automator: you can open it in Launchpad. Just type the name in your search bar to get it (or you can navigate it in Other folder).

Choose File->New. Here you need to choose a type of document to make. This is a decision of how your workflow would run. Here we choose service.

Select files or folders in Finder.app because we want to get the path in Finder.

Then focus on the Search Bar on your left hand and type "copy".

Then select and drag the Copy to Clipboard to the work area(right big area).

Click File->Save to save it(I named it as Copy Path) and this simple service has been completed.

Let's set up Reveal Finder Items before we test it.

Create a new service and select text in any application use the same method mentioned before and you will get an empty workflow.

Type Reveal in the Search Bar, and Drag the Reveal Finder Items to the right big area.

Save it(I saved as Reveal Items) and this service has been completed.

Then let's test these two functions. Open finder and select a file/folder as I do.

Then right click on it and choose Service->Copy Path. Then this file/folder's path has already been copied to your clipboard.

Now you can paste it anywhere.

Let's test the Reveal Items function now.

Highlight a path.

Right click on it, and choose Service->Reveal in Finder.

Then the item in your Finder will show up.



Finally

The other two examples will show up in the next blog. If you are interested in Automator, try it yourself!

There is no requirement to programming experience, but it will save you a lot of time.

Anyway, it's cool right?
