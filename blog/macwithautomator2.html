Mac with Automator : Part 2 by Steven Zhu

In the last blog I introduced a small example to show how Automator work. In this blog I'm gonna give another example to illustrate it more deeply.

This example is: Save and Clean the Desktop.



Save and Clean the Desktop

Most of the Mac users prefer to own an empty elegant Desktop.

However sometimes it is unavoidable to put some temporary files on your Desktop, such as when illustrating an example to your colleague, or some important files that you do not want to forget, or even some screen shot. And in some cases, we are not sure if those files can be deleted because we may use the files again in the near future.

So absolutely we need a mechanism to help us save the files and clean the Desktop automatically.

Before we start, I'll briefly introduce you how this machanism do. I'll build it as a service that you can use this service in any application. Once you click on this service, the Desktop will be cleaned and the files will be saved in a folder named after the current time. This folder would be saved in the location: Documents/History_Desktops/.

So let's begin.

Open Automator.

Choose File->New. Choose service.

This time we select no input in any application because we want to clean the Desktop whenever we want.

Then search New Folder and drag the New Folder to the right.

After that we need to generate a folder name automatically based on the current time. Click on the Variables and search today.

Then drag the Today's date to the Name blank field in the New Folder action on the right.

Use the same method to search and drag Current Time variable to the blank field after Today's date. Then right click the two elements to edit the format as Custom format...(if you do not choose Custom format..., this workflow may not work properly).

Because we want this generated folder to be put into the path: Documents/History_Desktops, we need to new a folder called History_Desktops in Documents manually. So head to your Documents folder and new a folder called History_Desktops there.

After that, back to your Automator and select this created folder in where field(you should choose other to locate your History_Desktops folder).

Then you will get one like this.

Then click Actions on the left and search set value, and drag the Set Value of Variable to the right. In the Drop-down menu of Variable in Set Value of Variable, choose New variable... and name it as Destination. Then you have set up your saving location.

It is time to wrap all files on the Desktop and move them into this destination.

Search get value and drag Get Value of Variable to the right. But you need to right click on this action to select Ignore Input because getting content in Desktop has no relationship with the last step.

Click on Variables on the left and search path, then drag the variable Path into the Drop-down menu in the action Get Value of Variable.

Click on the second button at the bottom main area, you will get variable panel(shown in image1). Then double click on the Path variable and change it to Desktop, remenber to use the Drop-down menu to select Desktop too(shown in image2).

Highlight a path.

Then select Action on the left and search get folder contents. Drag Get Folder Contents to the right and put it after Get Value of Variable.

After that, search move finder items, and drag the Move Finder Items to the right. Drag the destination variable to the Drop-down menu in Move Finder Items action.

Save it(I saved it as Save and Clean up Desktop).

Now let's test it.

In any application, you just need to click on the App name on the top bar and choose Services->Save and Clean up Desktop.

Now your Desktop has been cleaned up and all the files have been saved into Documents/History_Desktops!



Finally

The last example will show up in the next blog. If you are interested in Automator, try it yourself!

There is no requirement to programming experience, but it will save you a lot of time.
