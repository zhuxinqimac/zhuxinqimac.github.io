How to Install PostgreSQL by Steven Zhu

This is a blog for my own convenience to install PostgreSQL in case I forget it. This guidance is basically the same as the one provided by the course of Database System at UNSW.



Add Environment (Optional)

Create an environmental file "env" and copy these lines in it:

PGHOME=/where/you/want/to/put/psql

export PGDATA=$PGHOME/data

export PGHOST=$PGDATA

export PGPORT=54321

export LD_LIBRARY_PATH=$PGHOME/lib

export PATH=$PGHOME/bin:$PATH

Then append this line to ~/.bash_profile :

source /where/you/put/env

Then source your bash_profile.



Naive Installation

Linux: apt-get install postgresql

macOS: brew install postgresql



Initiate Database

You can initdb now, just type:

$ initdb

Then edit $PGDATA/postgresql.conf file.

unix_socket_directories = '/where/you/put/pgsql/data'

Then you can start your PostgreSQL server with:

$ pg_ctl start -l $PGDATA/log

And you can stop it by typing:

$ pg_ctl stop



Manual Installation

Download the PostgreSQL source code from official website.

Extract it:

$ tar xfj postgresql-9.3.3.tar.bz2

You will get a directory of the same name.

Compile it:

$ cd postgresql-9.3.3

$ ./configure --prefix=/where/you/want/to/put/pgsql

$ make

Then you get: All of PostgreSQL successfully made. Ready to install.

Then install it.

$ make install

Then you get: PostgreSQL installation complete.

Then set the environment as above.

Then initdb as above.
