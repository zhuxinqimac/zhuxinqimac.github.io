Mac with Automator : Part 3 by Steven Zhu

In this blog I'll use Automator to help you apply a filter to your images.


Add a Filter to Images

Some people like to apply filters to images. Such as...

However some of these conversions are very simple, and some would be repetitive. It would be very costly for human to do such kind of small jobs so many times. If we can find a robot to do these boring things for us, that would be wonderful.

So today we will use Automator to finish it.

Let's begin.

Open Automator.

Choose File->New. Choose Application(this time we use application as an example).

When using Application, the files dragged on it are considered as input.

Search on the Search Bar: filter, and drag Apply Quartz Composition Filter to Image Files to the right area.

You will receive a warning prompt, because your action may modify the original file. Just click Add.

Now you will get a Copy File Items action before your transmission action.

Choose Desktop in the Drop-down menu to make sure the modified images are stored on the Desktop.

Click on the Option button of Apply Quartz Composition Filter to Image Files and check Show this action when the workflow runs to make this window show when you are using this workflow because you may want to choose which type of conversion to be implemented(you may even make the last action show to change the save location).

Save it(I saved it as Apply Filter to Images.app) in Desktop folder.

Now let's have a test.

Drag an image on the app we created.

Choose which type of filter to apply(I chose ASCII Art). You can modify the parameters on the right panel.

TA-DA...

If you want the background to be black, you can add one more Apply Quartz Composition Filter to Image Files action in your application and choose Color Invert when you select the effect. Here I just drag the converted image to the App we created again to do it.

TA-DA...

You can drag multiple images to the App as well.

And I chose X-Ray...

If you want to know more, just try it yourself! It supports about 30 kinds of conversion. However, this is not a professional level of image modification. If you need more accurate and precise modification, please use more professional softwares.



Finally

If you are interested in Automator, try it yourself!

There is no requirement to programming experience, but it will save you a lot of time.

Anyway, it's cool right?



P.S. Wish Automator makes your life better.
